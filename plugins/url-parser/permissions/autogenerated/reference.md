## Default Permission

Default permissions for the url-parser plugin

#### This default permission set includes the following:

- `allow-parse-url`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`url-parser:allow-parse-url`

</td>
<td>

Enables the parse_url command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`url-parser:deny-parse-url`

</td>
<td>

</td>
</tr>
</table>
