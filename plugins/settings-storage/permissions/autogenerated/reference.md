## Default Permission

Default permissions for the settings-storage plugin

#### This default permission set includes the following:

- `allow-get-value`
- `allow-update-value`
- `allow-remove-value`
- `allow-batch-update-value`
- `allow-batch-get-value`
- `allow-batch-remove-value`
