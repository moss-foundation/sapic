## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-get-item`
- `allow-put-item`
- `allow-remove-item`
- `allow-batch-get-item`
- `allow-batch-put-item`
- `allow-batch-remove-item`
- `allow-batch-get-item-by-prefix`
- `allow-batch-remove-item-by-prefix`
